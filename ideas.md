## แนวทางที่ 3: Premium Fitness App — Dark Slate + Amber (CHOSEN)

**Design Movement:** Premium Mobile-First Dark UI + Warm Metallic Accents

**Core Principles:**
- Dark slate background สร้างความรู้สึก premium และ focused
- Amber/gold accent สื่อถึง achievement, excellence
- Layout แบบ card-based ที่ scroll ได้ลื่น เหมือน native app
- ข้อมูลแสดงชัดเจน ลำดับความสำคัญชัด

**Color Philosophy:**
- Background: Deep Slate #0F172A (slate-900)
- Surface: #1E293B (slate-800)
- Primary Accent: Amber #F59E0B
- Secondary: Teal #14B8A6
- Text Primary: #F1F5F9
- Text Muted: #94A3B8

**Layout Paradigm:**
- Top navigation bar แบบ sticky
- Hero section แบบ asymmetric ซ้ายคือ profile/stats ขวาคือ visual
- Weekly schedule แบบ horizontal scroll cards
- Workout detail แบบ accordion expand

**Signature Elements:**
- Amber glow effect บน active day/exercise
- Gradient text สำหรับ section headers
- Subtle grid pattern background texture
- Progress bars แบบ animated

**Interaction Philosophy:**
- Hover: subtle scale + glow
- Click: ripple effect
- Scroll: fade-in entrance animation
- Active state: amber highlight

**Animation:**
- Entrance: fade-up 200-300ms ease-out
- Cards: stagger 50ms per item
- Progress bars: animate on mount
- Hover: 150ms scale(1.02) + shadow

**Typography System:**
- Display: Oswald (bold, condensed, athletic)
- Body: Nunito Sans (friendly, readable)
- Numbers: Oswald Semibold
